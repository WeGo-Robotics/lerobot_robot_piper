+ CAN포트를 다 찾고 초기화하고 접속한다.
+ PIPER를 찾는다. SDK를 이용해서 연결하고 파이퍼가 연결된 CAN이 맞는지 검사한다.
+ 파이퍼가 리더/팔로워 모드 인지 확인한다.
+ CAN 포트의 이름을 변경할 건데, 사용자가 PIPER의 구성 형태를 선택한다.
    - 리더 1 / 팔로워 1
    - 팔로워 2
    - 리더 2
    - 리더 2 / 팔로워 2
+ PIPER의 구성 형태대로 리스트가 만들어지고 사용자는 각 PIPER에 있는 찾기 버튼을 누른다.
+ 찾기 중으로 변경되면 사용자는 로봇팔을 45도 이상 크게 움직인다.
+ 프로그램에서는 새 이름이 할당되지 않은 PIPER들의 관절 값을 모니터링 하다가 크게 이동한 PIPER을 찾으면 해당 PIPER에 새 이름을 할당한다.
+ 새 이름 할당이 완료되면 포트 설정을 저장한다.
+ 새 이름으로 CAN포트를 다시 초기화 한다.
